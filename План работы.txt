 Цель плана - создать многофункциональную веб-страницу которая может:
                1) Расчеты (желательно калькулятор)
                2) Заметки (возможность их создавать и открывать)
                3) Различные подсказки (справки по материалу и тд)

1. Определиться с редактором HTML - Выполнено (VS Code)
2. Создать index.html ввиде главной страницы. - Выполнено
3. Создать index.css для оформления страницы второго пункта.
4. Написать страницу index.html - В процессе
5. Написать оформление index.html в файле index.css
6. Создать новые страницы и оформить их.
7. Создать код для каждой функциональной страницы.
